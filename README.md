# userscripts

Personal [Tampermonkey](https://www.tampermonkey.net/) userscripts — mostly dark themes
plus small quality-of-life tweaks — for a handful of sites. Each site lives in its own
folder; install a userscript manager (Tampermonkey / Violentmonkey) and load the script.

| Folder | Script | Site | Version |
| ------ | ------ | ---- | ------- |
| [`kemono.su/`](kemono.su/) | Kemono Tools | kemono.su / kemono.cr | 0.4.0 |
| [`bilibili.com/`](bilibili.com/) | Bilibili Fixes | bilibili.com | 0.1.0 |
| [`rule34.xxx/`](rule34.xxx/) | Dark Theme | rule34.xxx | 0.1.6 |
| [`v3rmillion.rip/`](v3rmillion.rip/) | Dark Theme | v3rmillion.rip | 0.1.4 |
| [`youtube.com/`](youtube.com/) | Dark Theme | youtube.com | 0.1.4 |
| [`unknowncheats.me/`](unknowncheats.me/) | Dark Theme | unknowncheats.me | 0.1.0 |

`kemono.su/` is the most fully-featured (theme + collapsible sidebar + per-creator
blacklist + URL linkifier) and has its own [README](kemono.su/README.md); the rest are
primarily CSS themes. Assets for each script live in that folder's `_icons/`.
